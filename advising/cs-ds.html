DS/CS DOUBLE MAJOR

BACHELORS OF SCIENCE IN DATA SCIENCE AND COMPUTER SCIENCE

Five (5) courses in Computer Science, six (6) courses in Data Science, one (1) course in Mathematics.

  1. CS   151 : Intro to Programming in Python
  
  2. CS   152 : Data Structures
  
  3. DATA 151 : Intro to Data Science in R
  
  4. DATA 152 : Inferential Statistics
  
  5. MATH 251W: Foundations of Advanced Mathematics
  
  6. DATA 251 : Math for Data Science*
  
  7. DATA 252 : Models and Machine Learning
  
  8. CS   261 : Software Development
  
  9. CS   271 : Networks and Systems
  
  10. CS   351 : Analysis of Algorithms
  
  11. DATA 351 : Data Management with SQL
  
  12. DATA 352W: Ethics, Teamwork, Communications


*Requires MATH 152: Accelerated Calculus II or High School equivalent.

Dependency Graph

Some notes:

  1. CS 151 serves as a sufficent but not necessary pre-req for MATH 251W, CS 152, DATA 351, and DATA 352W.
    • For MATH 251W any 100-level CS, DATA, or MATH class would suffice.
    • For DATA 352W, CS 151 or DATA 151 would suffice.
    • For CS152 and DATA 351, CS 151 is required.
  2. DATA 251 has a calculus requirement.
  3. There are no remaining elective requirements.
